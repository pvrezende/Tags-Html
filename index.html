Surgimento da Internet

A internet surgiu na década de 1960 como um projeto de pesquisa do Departamento de Defesa dos Estados Unidos, conhecido como ARPANET. O objetivo inicial era criar uma rede de comunicação resistente a falhas, capaz de conectar computadores em diferentes locais. Com o tempo, a internet evoluiu e se tornou uma ferramenta essencial para a comunicação, compartilhamento de informações e acesso a serviços em todo o mundo.

Os hipertextos são elementos fundamentais da internet, permitindo a navegação entre páginas web através de links.

Ao longo dos anos, a internet se expandiu e se tornou uma parte integrante da vida cotidiana, transformando a forma como nos comunicamos, trabalhamos e nos divertimos. Ao visitar um site, o navegador interpreta o código HTML para exibir o conteúdo de forma visual e interativa, permitindo que os usuários acessem informações, realizem compras online, se conectem com outras pessoas e muito mais. A internet continua a evoluir, trazendo novas tecnologias e oportunidades para a sociedade global.

E assim, a internet se tornou uma parte indispensável da vida moderna.

Como funciona o HTML

O HTML (HyperText Markup Language) é a linguagem de marcação utilizada para criar páginas web. Ele define a estrutura e o conteúdo de uma página, utilizando elementos chamados de tags. Cada tag tem um propósito específico e é usada para formatar o texto, inserir imagens, links e outros elementos.

Foi criado por Tim Berners-Lee em 1991, enquanto trabalhava no CERN, o Laboratório Europeu de Física Nuclear. Onde seu talento se manifestou na criação da primeira página web.

A participação de Tim Berners-Lee na criação do HTML foi fundamental para o desenvolvimento da web moderna. Pois ele estabeleceu as bases para a comunicação e o compartilhamento de informações na web.

O que são Tags

As tags são os elementos básicos do HTML que definem a estrutura e o conteúdo de uma página web. Elas são usadas para formatar o texto, inserir imagens, criar links e muito mais. As tags são escritas entre colchetes angulares (< >) e geralmente vêm em pares, com uma tag de abertura e uma tag de fechamento.

Por exemplo, a tag <h1> é usada para criar um título de nível 1, enquanto a tag <p> é usada para criar um parágrafo. As tags permitem que os desenvolvedores web criem páginas visualmente atraentes e organizadas, facilitando a navegação e a compreensão do conteúdo pelos usuários.

Além disso, as tags também podem conter atributos, que fornecem informações adicionais sobre o elemento. Por exemplo, a tag <a> é usada para criar um link, e o atributo href especifica o destino do link. As tags são essenciais para a construção de páginas web e desempenham um papel crucial na criação de uma experiência de usuário agradável e funcional.

Existem mais tags diferentes que vou sitar abaixo e explicar cada uma :

  • <div>: Usada para agrupar elementos e criar seções em uma página web.

    • exemplo: <div><h1>Título</h1><p>Parágrafo</p></div>

  • <header>: Usada para definir o cabeçalho de uma página ou seção.

    • exemplo: <header><h1>Título</h1></header>

  • <span>: Usada para aplicar estilos a uma parte específica do texto.

    • exemplo: <span style="color: red;">Texto vermelho</span>

  • <img>: Usada para inserir imagens em uma página web.

    • exemplo: <img src="imagem.jpg" alt="Descrição da imagem">

  • <ul>: Usada para criar uma lista não ordenada.

    • exemplo: <ul><li>Item 1</li><li>Item 2</li></ul>

  • <ol>: Usada para criar uma lista ordenada.

    • exemplo: <ol><li>Item 1</li><li>Item 2</li></ol>

  • <li>: Usada para definir um item em uma lista.

    • exemplo: <li>Item 1</li>

  • <table>: Usada para criar tabelas em uma página web.

    • exemplo: <table><tr><td>Célula 1</td><td>Célula 2</td></tr></table>

  • <tr>: Usada para definir uma linha em uma tabela.

    • exemplo: <tr><td>Célula 1</td><td>Célula 2</td></tr>

  • <td>: Usada para definir uma célula em uma tabela.

    • exemplo: <td>Célula 1</td>
